Uvod u HTML

HTML - HyperText Markup Language

HTML je osnovni gradivni element svakog sajta, definiše značenje i strukturu veb sadržaja.

HTML Dokument

Svaki html dokument počinje sa !DOCTYPE deklaracijom koja govori koji je tip dokumenta.

Deklaracija html5 fajla izgleda ovako:

<!DOCTYPE html>

<html>

Sam HTML dokument počinje sa <html> i završava se sa </html>

<!DOCTYPE html>

<html>



/* html dokument */



</html>

<head>

Meta podaci poput SEO optimizacija, naslova sajta i charset-a html dokumenta stavljaju se u <head> element

<!DOCTYPE html>

<html>


<head>

<meta name="keywords" content="HTML, CSS, JavaScript">

<meta charset="UTF-8"/>

<title>naziv sajta</title>

</head>



</html>

<body>

Vidljivi sadržaj stranice se stavlja u <body> element

<!DOCTYPE html>

<html>


<body>

<h1>novi naslov</h1>

<p>novi paragraf</p>

<img src="mojaSlika.png" wih="50px" height="50px">

</body>



</html>

Šablon sajta

Standardni šablon html sajta izgleda ovako:

<!DOCTYPE html>

<html>

<head>


/* meta podaci */


</head>


<body>


/* vidljivi elementi */


</body>

</html>
