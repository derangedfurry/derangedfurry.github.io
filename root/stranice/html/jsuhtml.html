JavaScript u HTML-u

JavaScript

JavaScript služi za uvođenje dinamičkih i interaktivnih elemenata u HTML.


oznaka <script> se koristi za definisanje JavaScript skripte.


Uobičajena upotreba JavaScript-a su manipulacija slikama, validacija forme i dinamičke promene sadržaja.


Da bi izabrao HTML element, JavaScript najčešće koristi metodu document.getElementById().

<!DOCTYPE html>

<html>

<head>


<link href="stil.css">


</head>


<body>


<h1>Naslov</h1>


<p id="paragraf">Paragraf</p>

<button onclick="Funkcija()">Klikni me</button>


</body>

<script>

function Funkcija() {

document.getElementById("paragraf").style.fontSize = "25px";

document.getElementById("paragraf").style.color = "red";

document.getElementById("paragraf").style.backgroundColor = "yellow";

}

</script>

</html>

Ekstreni JavaScript

Ekstreni JavaScript podrazumeva korišćenje src atributa kod <script> elementa.

<script src="Skripta.js">

</script>
