CSS u HTML-u

CSS

CSS (Cascading Style Sheet), Pomoću CSS-a mogu se kontrolisati boja, font, veličina teksta, razmak između elemenata, kako su elementi pozicionirani i postavljeni, koje pozadinske slike ili boje pozadine treba da se koriste, različite ekrane za različite uređaje i veličine ekrana i još mnogo toga!

Korišćenje CSS-a

CSS se može dodati u HTML dokumente na 3 načina:

  • Inline
  • Interni
  • Eksterni

Inline CSS

Inline CSS se koristi za primenu jedinstvenog stila na jedan HTML element.


Inline CSS koristi atribut stila HTML elementa.

<h1 style="color:blue;">Naslov</h1>

<p style="color:red;">Paragraf</p>

Naslov

Paragraf

Interni CSS

Interni CSS se koristi za definisanje stila za jednu HTML stranicu.

Interni CSS je definisan u <head> elementu HTML stranice, unutar elementa <style>.

<!DOCTYPE html>

<html>

<head>


<style>


h1 {background-color: gray; color: white;}


p {background-color: lightblue; color: purple;}


</style>


</head>


<body>


<h1>Naslov</h1>


<p>Paragraf</p>


</body>

</html>

Paragraf

Eksterni CSS

Eksterni CSS se koristi za definisanje stila za mnoge HTML stranice.


Da bi se koristio eksterni CSS, dodaje je se veza do njega u <head> elementu svake HTML stranice.

<!DOCTYPE html>

<html>

<head>


<link href="stil.css">


</head>


<body>


<h1>Naslov</h1>


<p>Paragraf</p>


</body>

</html>

stil.css

Eksterni CSS se može napisati u bilo kom tekst editoru. Datoteka ne sme da sadrži nikakav HTML kod i mora biti sačuvana sa ekstenzijom .css.


h1 {
background-color: lightyellow;
color: green;
}


p {
background-color: darkblue;
color: white;
}
