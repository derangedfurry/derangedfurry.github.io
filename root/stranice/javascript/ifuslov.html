JavaScript uslovno izvršavanje

JavaScript uslovni iskazi

JavaScript sadrži pet uslovnih iskaza za kontrolu toka izvršavanja koda.

  • if

  • if...else

  • if...else if...else

  • switch

  • ternarni operator (? :)

if

if izvršava blok koda ako je uslov tačan.

if (uslov) {

//kod koji će se izvršiti

}

if...else

U slučaju da je uslov netačan, izvršava se blok koda u else iskazu.

if (uslov) {

//kod koji će se izvršiti ako je uslov tačan

} else {

//kod koji će se izvršiti ako je uslov netačan

}

if...else if...else

Uslov se proverava u prvom if iskazu, ako je netačan, proverava se u sledećem iskazu.

if (uslov1) {

//kod koji će se izvršiti ako je uslov1 tačan

} else if (uslov2){

//kod koji će se izvršiti ako je uslov2 tačan

}...

else {

//kod koji će se izvršiti ako nijedan uslov nije tačan

}

switch

switch se koristi kod višestrukog grananja.

switch(uslov1) {

case x:

//blok koda

break;

case y:

//blok koda

break;

case z:

//blok koda

break;

...

case n:

//blok koda

break;

default:

//blok koda

}

ternarni operator

Skraćena verzija if...else uslova.


U slučaju da je uslov tačan, izvršava se iskaz1, ako ne onda se izvršava iskaz2.

uslov ? iskaz1 : iskaz2 ;
