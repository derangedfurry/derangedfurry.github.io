JavaScript

JavaScript je programski jezik veb stranica


Može da izračuna, manipuliše i potvrdi (validira) podatke.


Može da ažurira i menja HTML i CSS veb stranice.


U HTML-u, JavaScript kod se ubacuje između oznaka <script> i </script>.

<script>

document.getElementById("primer").innerHTML = "JavaScript omogućuje izmenu elemenata!"

</script>
