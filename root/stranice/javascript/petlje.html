Petlja

Petlje su korisne ako želite da pokrećete isti kod iznova, svaki put sa drugom vrednošću.


umesto:

tekst += lista_imena[0] + "<br/>"

tekst += lista_imena[1] + "<br/>"

tekst += lista_imena[2] + "<br/>"

tekst += lista_imena[3] + "<br/>"

tekst += lista_imena[4] + "<br/>"

tekst += lista_imena[5] + "<br/>"

može se napisati:

for (let i = 0; i < lista_imena.length; i++) {

tekst += lista_imena[i] + "<br/>";

}

JavaScript sadrži nekoliko petlji:

  • for

  • while

  • do while

for

for petlja se satoji iz tri dela:


for ( brojač ; uslov ; korak )


brojač - se izvršava jednom pre početka for petlje.

uslov - proverava se uslov for petlje pri svakoj iteraciji, ako je tačan treće sledeća iteracije, ako nije, onda se prekida for petlja.

korak - izvršava se na kraju svake iteracije, obično je to inkrementacija brojača.

let suma = 0;

for (let i = 0; i < 5; i++) {

suma += i;

}

while

while petlja prolazi kroz blok koda sve dok je određeni uslov tačan.

while (uslov) {

//kod koji će se izvršiti ako je uslov tačan

}

let suma = 0;

while (i < 10) {

suma += i;

i++;

}

do while

do while je varijanta while petlje.

do while prvo izvršava blok koda onda provera uslov.

do {

//kod koji će se izvršiti

}

while (uslov);

let suma = 0;

let i = 1;

do {

suma += i;

i++;

}

while (i < 10);
