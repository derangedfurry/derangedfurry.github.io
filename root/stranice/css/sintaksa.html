Sintaksa

CSS pravilo se sastoji od selektora i bloka deklaracije:


Selektor {osobina : vrednost ;}

Primer 1

h2 {font-family : Verdana ;}

Primer 2

body {

display : flex ;

flex-direction : column ;

height : 100vh ;

}

Selektor pokazuje na HTML element koji se stilizuje.


Blok deklaracije sadrži jednu ili više deklaracija razdvojenih tačkom i zarezom.


Više CSS deklaracija je odvojeno tačkom i zarezom, a blokovi deklaracija su okruženi vitičastim zagradama.
