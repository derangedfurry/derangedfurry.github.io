CSS osnove

CSS

CSS (Cascading Style Sheet) je jezik koji se koristi za stilizovanje veb stranice.


CSS se koristi za definisanje stilova veb stranica, uključujući dizajn, izgled i varijacije u prikazu za različite uređaje i veličine ekrana.


CSS definicije se obično čuvaju u spoljnoj .css datoteci.


Primer.css

body {
background-color: lightblue;
}


h1 {
background-color: lightyellow;
color: green;
}


p {
font-family: Roboto;
font-size: 1.5em;
}
