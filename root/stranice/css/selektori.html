CSS selektori

Selektori

CSS selektori se koriste za „pronalaženje“ (ili odabir) HTML elemenata koji će se stilizovati.


CSS selektori se mogu podeliti u pet kategorija:

  • Jednostavni selektori (odabir elemenata na osnovu imena, ID-a, klase)

  • Kombinatorni selektori (odabir elemenata na osnovu specifičnog odnosa između njih)

  • Selektori pseudoklasa (odabir elemenata na osnovu određenog stanja)

  • Selektori pseudoelemenata (odaberite i stilizujte deo elementa)

  • Selektori atributa (odabir elemenata na osnovu atributa ili vrednosti atributa)

Najčešće korišćeni selektori su selektori id, klase i univerzalni selektor

ID selektor

ID selektor koristi atribut id HTML elementa da bi izabrao određeni element.


ID elementa je jedinstven unutar stranice, tako da se id selektor koristi za odabir jednog jedinstvenog elementa.


Da bi se selektovao specifični id, koristi se # karakter, praćen id-jem elementa.

#id-elementa {

osobina : vrednost ;

...

}

Selektor klase

Selektor klase bira HTML elemente sa određenim atributom klase.


Da bi se selektovala specifična klasa koristi se . karakter, praćen imenom klase.

.ime-klase {

osobina : vrednost ;

...

}

Moguće je odrediti na koje elemente sa tom klasom će uticati izmene.

p.ime-klase {

osobina : vrednost ;

...

}

Univerzalni selektor

Univerzalni selektor koji koristi * karakter, uzima sve HTML elemente na stranici u obzir.

* {

osobina : vrednost ;

...

}

Selektor grupisanja

Selektor grupisanja bira sve HTML elemente sa istim definicijama stila.

h1 {

color : red ;

font-family : Roboto ;

}

p {

color : red ;

font-family : Roboto ;

}

a {

color : red ;

font-family : Roboto ;

}

U primeru se mogu primetiti da elementi imaju identične definicije stila


Selektor grupisanja se može primeniti pri minimiziranju količine koda, primenom ( , ) između elemenata.

h1, p, a {

color : red ;

font-family : Roboto ;

}
